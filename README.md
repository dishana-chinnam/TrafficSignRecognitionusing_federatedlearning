# TrafficSignRecognitionusing_federatedlearning
A deep learning-based Traffic Sign Recognition (TSR) system built using Federated learning, TensorFlow and Convolutional Neural Networks (CNN). This project classifies traffic signs from image datasets and demonstrates image preprocessing, model training, evaluation, and prediction.
## 📂 Project Structure
TSR_Project/ │── TSR_tf.ipynb # Main notebook │── requirements.txt # Required dependencies │── environment.yml # Conda environment file │── README.md # Project documentation │── dataset/ # Dataset folder │── model/ # Saved model files

---

📊 Model Workflow
Load and preprocess dataset
Normalize image data
Build CNN model using TensorFlow
Train the model
Evaluate accuracy and loss
Predict traffic sign classes
